Image Instructions

For posts;

* Hero images must have a MINIMUM aspect ratio of 16:9 (1.778)
  - While the width of images is fixed (max. 1920px), the height can be changed (made shorter) if needed.
  - This will make the aspect ratio LARGER.

* Content images can be any size, but ideally will have an aspect ratio of 16:9
  - While the images can be different sizes, they must have the same aspect ratio if two or more are used.

* Thumbnails MUST have a 16:9 ratio.
  - The ideal size is 350 x 197.

* The maximum width of images are;
  - HERO: 1920 x 1080
  - CONTENT: 1024 x 576
  - THUMBNAILS: 350 x 197
